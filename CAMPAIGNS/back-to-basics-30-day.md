# 30-Day Back to Basics Campaign

Date: 2026-05-28

## Goal

Create a simple 30-day practice message for ReadEasy30 and MathEasy30.

## Message

Thirty days. One calm step each day.

## Audience

- Parents
- Adult learners
- ESL learners
- Tutors
- Homeschool helpers
- Community groups

## Simple Routine

1. Read for a short time.
2. Practice one math skill.
3. Review calmly.
4. Repeat tomorrow.

## Page Ideas

## Social Ideas

1. Day 1 starts with one calm step.
2. Do not rush the learner.
3. Small practice is still practice.
4. Review mistakes without shame.
5. Return tomorrow.
