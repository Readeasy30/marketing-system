# Education Library / Nonprofit Active Outreach Plan

Purpose: reach libraries, adult literacy groups, ESL programs, tutoring groups, community centers, churches, nonprofits, and workforce-development helpers with free ReadEasy30 and MathEasy30 resources.

## Outreach job

This is not a sales campaign. This is resource-sharing and feedback collection.

Best targets:

- public libraries
- adult literacy nonprofits
- ESL programs
- community education groups
- tutoring centers
- homeschool co-ops
- workforce-development nonprofits
- churches or community groups with education programs
- senior centers with adult learning support

## Core rules

- Be clear that the resources are free public practice pages right now.
- Do not claim guaranteed results.
- Do not claim to replace teachers, tutors, schools, specialists, or assessments.
- Ask for feedback.
- Keep emails short.
- Track every contact.
- Do not mass spam.

| Day | Action | Goal |
|---|---|---|
| 1 | Build list of 10 local/regional libraries | Start small |
| 2 | Build list of 10 adult literacy or ESL groups | Find aligned targets |
| 3 | Send 3 library emails | Test wording |
| 4 | Send 3 adult literacy/ESL emails | Test adult learner wording |
| 5 | Send 3 tutor/community education emails | Test worksheet value |
| 6 | Record replies and objections | Improve message |
| 7 | Pause outreach and review | Avoid spam |
| 8 | Send 5 improved emails | Use better wording |
| 9 | Send 5 more emails | Continue small batch |
| 10 | Ask 2 contacts for feedback | Learn what is missing |
| 11 | Create FAQ improvements from replies | Site improvement loop |
| 12 | Send follow-up only to warm replies | No cold nagging |
| 13 | Record best contact type | Choose next target |
| 14 | Review and decide next outreach batch | Evidence-based next step |

## Review after 14 days

- Which type of organization replied?
- Which resource got the most interest?
- Did adult reading, ESL, worksheets, or math get the most response?
- What objections came up?
- What page should be improved next?
- Should a one-page PDF/resource sheet be created?

## Decision rule

If libraries or nonprofits respond positively, create a simple resource handout page and a printable one-page guide. Do not scale outreach until the message is proven.
