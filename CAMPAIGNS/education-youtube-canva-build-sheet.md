# Education YouTube Shorts Canva Build Sheet

Use this file to build the first 6 YouTube Shorts fast in Canva, HeyGen, CapCut, or another simple video tool.

Related plan:

- `CAMPAIGNS/education-youtube-2-week-plan.md`

## Global video settings

- Format: vertical 9:16
- Length: 30-40 seconds
- Posting pace: 3 times per week
- Batch size before review: 6 videos
- Background: calm, clean, light
- Text: large, high contrast, easy to read on phone
- Voice: slow, warm, clear
- Music: optional, very low volume
- CTA: one site link only per video when possible

## Reusable 5-scene structure

Use the same structure for all six videos:

| Scene | Time | Purpose | Visual |
|---|---:|---|---|
| 1 | 0-4 sec | Hook | Big text headline |
| 2 | 4-12 sec | Problem | Simple icon or calm learner image |
| 3 | 12-24 sec | Solution | Site screenshot, worksheet, or simple text cards |
| 4 | 24-33 sec | Benefit | Calm outcome wording |
| 5 | 33-40 sec | CTA | Site name + link |

## Brand style

### ReadEasy30

- Colors: soft teal, blue, cream, white
- Mood: calm, patient, no shame
- Mascot/character: Bubbles
- Use words: calm, read slowly, short lesson, no shame, find proof
- Avoid words: cure, guarantee, diagnosis, special-ed result, fast fix

### MathEasy30

- Colors: soft blue, teal, cream, white
- Mood: calm, confidence-building, one step at a time
- Mascot/character: Bubbles math coach
- Use words: one small step, hints, drawings, confidence first, slow practice
- Avoid words: genius, ace tests, guaranteed, instant improvement
