## Purpose

Pinterest can work well for evergreen education content.

Use these pin titles, descriptions, and destination links for the first ReadEasy30 + MathEasy30 promotion push.

## Pinterest rules

- Use vertical pins.
- Keep text large.
- Use one clear idea per pin.
- Link each pin to the most relevant page.
- Do not use exaggerated claims.
- Reuse winning pin topics with new designs.
