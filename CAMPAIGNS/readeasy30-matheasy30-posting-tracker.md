## Weekly review questions

Answer these once a week:

1. Which topic got the most comments?
2. Which link got the most clicks?
3. Which audience responded: parents, adult learners, ESL learners, tutors, homeschool, or local community?
4. What question came up more than once?
5. What page should be created or improved next?
6. What graphic style looked best?
7. Which post should be reused with a better headline?

## Simple rules

- Do not chase every platform at once.
- Start with Facebook, Pinterest, and one short-video platform.
- Use LinkedIn for professional/helper positioning.
- Use Reddit carefully and only where helpful.
- Reuse the post that gets real response.
- Improve one page or post each week based on questions.
