# Canva Manual Export Steps

Last updated: 2026-06-03

Purpose: The Canva connector can create and find designs, but this workflow does not currently provide a reliable direct export action here. Use these manual export steps so campaign posting does not stall.

## First image export: Canva Card 1

Open:

https://www.canva.com/d/cAqhrbhNDjRWro6

Export as:

`readeasy30-matheasy30-card-1-calm-step.png`

Steps:

1. Open the Canva edit link.
2. Click **Share**.
3. Click **Download**.
4. Choose **PNG**.
5. Select the current page only if Canva asks.
6. Click **Download**.
7. Save using the suggested file name.
8. Post manually using `EDUCATION-POST-NOW-CHECKLIST.md`.
9. Paste the live post link into `CAMPAIGN-PUBLISHING-TRACKER.md`.

## First video export: Video 01

Open:

https://www.canva.com/d/-AsIThP__kPG-JB

Export as:

`readeasy30-matheasy30-video-01-homework-fight.mp4`

Steps:

1. Open the Canva edit link.
2. Click **Share**.
3. Click **Download**.
4. Choose **MP4 Video**.
5. Select all pages if Canva asks.
6. Click **Download**.
7. Save using the suggested file name.
8. Post manually using `EDUCATION-DAY-6-SHORT-VIDEO-PUBLISHING-PACKET.md`.
9. Paste the live post link into `CAMPAIGN-PUBLISHING-TRACKER.md`.

## Do not repeat work

If exporting fails:

1. Mark the item `Blocked` in `CAMPAIGN-PUBLISHING-TRACKER.md`.
2. Record the reason.
3. Move to the next prepared asset.

## Minimum win

One of these is enough:

- Card 1 exported as PNG.
- Video 01 exported as MP4.
- One live post link added to the tracker.
- One blocker clearly recorded.
