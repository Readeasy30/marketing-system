## Manual tracker update

Update `CAMPAIGN-PUBLISHING-TRACKER.md`.

Use these statuses:

- Published: live post exists
- Blocked: account, access, or route not ready
- Designed: asset exists but not exported or posted
- Drafted: copy exists only
- Paused: intentionally not using now
- Needs edit: asset exists but should be fixed before use
