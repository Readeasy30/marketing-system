## Weekly Review

Answer these every Sunday:

1. What was published?
2. What got visits, clicks, starts, or replies?
3. Which page/post should be improved?
4. Which site is blocked?
5. What is next week's main site?
6. What should AI produce next?

## Status Labels

## One-Person Rule

If a task takes too much manual effort, simplify it.

Prefer repeatable templates, AI drafts, and weekly batches over daily custom work.
