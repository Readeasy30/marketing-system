# Education Prospect Queue

Last updated: 2026-06-02

This queue supports ReadEasy30 and MathEasy30 outreach.

## Status labels

Use these labels exactly:

- `Direct verified public email`
- `Contact-form only`
- `Directory/source only`
- `Hold/media-only`
- `Blocked/no-contact`
- `Sent`
- `Submitted form`
- `Follow-up due`
- `Closed/no reply`

## Daily execution

Use `DAILY-EXECUTION-RULES.md`.

Recommended daily action:

- 5 contact forms per active outreach day.
- One segment per day.
- One follow-up only after 7 to 10 days.
- Stop on opt-out.

## Repair note

This queue intentionally separates contact-form prospects from direct-email prospects. A prospect is not email-ready until it passes `EMAIL-READY-CONTACT-STANDARDS.md`.
