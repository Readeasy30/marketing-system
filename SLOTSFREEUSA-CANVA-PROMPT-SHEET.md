# SlotsFreeUSA Canva Prompt Sheet

Last updated: 2026-05-28

## Purpose

Use these prompts to create simple Canva graphics for SlotsFreeUSA social posts, Pinterest pins, short video covers, and article thumbnails.

## Brand Direction

- dark navy background
- gold highlights
- clean casino-inspired style
- readable text
- no flashing clutter
- adult-focused
- beginner-friendly
- honest and safe

## Core Design Rules

- Use large text.
- Keep each graphic to one message.
- Avoid fake jackpot imagery.
- Avoid cash piles, income promises, or desperate gambling visuals.
- Use disclosure-friendly language.
- Make graphics readable on phones.

## Canva Prompts

### 1. Play Free First

Create a mobile-friendly social graphic with a dark navy background, gold accents, simple slot symbols, and large text that says: "Play Free First". Add small text: "Learn before offers." Style should feel clean, adult, and trustworthy.

### 2. Bonus Terms Checklist

Create a Pinterest-style checklist graphic with dark navy and gold styling. Title: "Casino Bonus Terms To Check". Checklist items: age rules, location rules, expiration date, playthrough rules, support options. Make it clear and easy to read.

### 3. Social Casino Coins Explained

Create a simple educational graphic showing two coin icons labeled "Fun Coins" and "Promotional Coins". Title: "Social Casino Coins Explained". Add small footer: "Read terms before signing up." Use a clean adult design.

### 4. Sweepstakes Rules To Check

Create a vertical Pinterest pin. Title: "Sweepstakes Casino Rules To Check". Include: coin type, location rules, age rules, redemption terms, support. Use simple icons and high contrast.

### 5. No Guaranteed Winnings

Create a responsible-play reminder graphic. Title: "No Guaranteed Winnings". Subtitle: "Slot-style games are entertainment, not income." Use calm design, no scary imagery.

### 6. How We Review Sites

Create a trust-focused graphic. Title: "How We Review Casino-Style Sites". Include: bonus clarity, age rules, location limits, mobile use, support, safer-play tools. Use clean card layout.

### 7. Free Slots Beginner Guide

Create a beginner-friendly graphic. Title: "Best Free Slots for Beginners". Include three points: simple symbols, free-play demo, read terms first. Use classic fruit slot symbols lightly.

### 8. Responsible Play Checklist

Create a simple checklist graphic. Title: "Responsible Play Checklist". Include: set a time limit, set a budget, do not chase losses, stop if stressed, use free play first.

## Text Safety

Do not use:

- win big
- guaranteed payout
- make money
- secret system
- risk-free gambling
- legal everywhere
- easy casino cash

Use instead:

- play free first
- read terms
- compare carefully
- learn the rules
- stay in control
- entertainment only
