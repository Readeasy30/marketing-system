# Avatar Embed Template

Use this as a safe planning template before adding any avatar or video embed to a website repo.

## Placement recommendation

Use one of these safe placements:

1. Homepage intro section below the hero.
2. Help card near the first call-to-action.
3. Dedicated guide page.
4. Small optional video block, not a forced popup.

## Embed rules

- Do not add API keys.
- Do not add secret tokens.
- Do not add live tracking scripts unless approved.
- Do not autoplay with sound.
- Do not block important page buttons.
- Do not make the avatar required to use the site.
- Keep the page useful without video.

## Safe HTML structure to create inside a website repo

Create a normal content card with:

- a short heading
- a short visitor benefit sentence
- a video area
- a button to the best next page

Add the actual provider embed only after the video provider is chosen and approved.

## Mobile QA checklist

- Video fits on small phone screens.
- Visitor can scroll past it easily.
- Buttons are still clickable.
- Page speed remains acceptable.
- The site still works if the video provider fails.
