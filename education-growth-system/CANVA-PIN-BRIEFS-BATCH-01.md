# Canva Pin Briefs Batch 01: ReadEasy30 + MathEasy30

Last updated: 2026-06-02

These briefs are for creating the first Pinterest-ready graphics in Canva.

## General design rules

- Keep text large.
- Use calm, clean layouts.
- Make designs readable on a phone.
- Avoid tiny text.
- Avoid clutter.
- Avoid childish designs that may turn away adult learners.
- Use friendly education visuals: books, worksheets, pencil, notebook, desk, soft icons.
- Keep wording respectful for adults, ESL learners, parents, tutors, and students.

## Canva creation checklist

Before exporting each pin:

1. Text readable on phone.
2. Website name visible.
3. No spelling mistakes.
4. No overpromising.
5. No medical, diagnostic, or guaranteed-result claims.
6. Looks respectful for adults and children.
7. Export as PNG.
8. Save final image name.
9. Record where it is posted in `DAILY-EXECUTION-TRACKER.md`.

## Pinterest posting checklist

For each pin:

1. Use the correct title.
2. Use the matching description from `FIRST-PROMOTION-BATCH-01.md`.
3. Link to the best matching page, not only the homepage.
4. Choose the most relevant board.
5. Record the post date and link.
