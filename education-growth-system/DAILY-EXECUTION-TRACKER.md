# Daily Execution Tracker: ReadEasy30 + MathEasy30

Last updated: 2026-06-02

Use this file to record actual marketing actions so work does not get repeated.

## Daily rule

Do a small number of useful actions and record them.

Quality beats volume.

## What to record

For every outreach or post, record:

- Date.
- Site promoted.
- Audience.
- Page/resource promoted.
- Channel.
- Exact organization/group/platform.
- Contact route used.
- Message type.
- Result.
- Follow-up date, if allowed.
- Blocker, if any.

## Weekly review

At the end of each week, answer:

1. Which page got the most useful response?
2. Which audience seemed most interested?
3. Which message sounded clearest?
4. Which page needs better title, intro, or call to action?
5. Which channel should be repeated?
6. Which route should be stopped?
